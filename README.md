## Available Scripts

In the project directory, you can run:

### `npm install`
Se debe instalar las dependencias y librerias existentes en el package.json para poder hacer uso de estas, ellas se guardaran en node_module.

### `npm start`
Este script hara una compilacion de los archivos js, dara seguimiento a los cambios que puedan haber y Ademas levantara un servidor local  [http://localhost:3000](http://localhost:3000).
